Pycharm 环境搭建及配置详解

发表于 2021-06-19 更新于 22021-06-19 分类于 Python
升级Python环境到3.8导致了一堆麻烦和调整，不过调整完成之后发现微调的代码运行速度倒是加速了50%左右，总算在一堆鸡皮算毛的微调之后来点慰藉。

PyQt5

最近升级Python版本发现pyqt5-tools==5.15.4.3.2更换了designer的安装位置到

然后pyuic的命令也发生变化了

新版本的qt-designer默认汉化了一部分操作界面，操作体验相当不爽了。

在目录XX\Miniconda3\Lib\site-packages\qt5_applications\Qt\translations使用designer_en.qm 替换designer_zh_CN.qm之后可以恢复英文界面。

自动添加文件头注释

文件头模板设置路径 Settings->Editor->File and Code Templates->Python Script
